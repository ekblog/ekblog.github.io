Cara Test Vps Disk Speed (I/O)

Jan 21, 2021 • One minute to read

Simpan snippet aja disini, biar aman nanti kalau butuh. Barusan salah satu cek thread di lowendtalk nemu command dibawah ini. Saya sih nyoba ga pake vps, tapi langsung dilaptop karena penasaran dengan speed SSD saya.

dd if=/dev/zero of=test bs=64k count=16k conv=fdatasync

Tadi di thread lowendtalk itu orangnya komplen kalau disk I/O itu sangat kecil ga cocok sama yang di promosikan yaitu menggunakan SSD.
