  • Canti Ganti Permalink Di Hugo

    Jul 26, 2021

    Default url untuk artikel di hugo adalah menggunakan tambahan post/posts, contoh artikel remove login walls di quora kemaren menggunakan namadomain.com/posts/url-artikel Bagaimana jika urlnya ingin seperti wordpress tanpa embel-embel post/posts lagi. Cara pertama adalah mengatur file config.toml atau config.yaml, sesuaikan dengan jenis konfigurasi yang dipakai. Saya biasanya menggunakan config.toml Tambahkan code [permalinks] posts = "/:title/" Untuk menggunakan title sebagai url atau [permalinks] posts = "/:filename/" Jika ingin menggunakan nama file sebagai alamat url.…more

  • Remove Login Walls Signup Di Quora

    Jul 22, 2021

    Cara ini manual ya, kalau mau yang otomatis tinggal cari ekstension untuk chrome atau firefox. Tambahkan ?share=1 di akhir url dan tekan enter Misal halamannya https://www.quora.com/bla-bla-bla jadi https://www.quora.com/bla-bla-bla/?share=1 Eh bisa juga pake ini ding, kalau udah ada plugin Ublock Origin di firefox/chrome tinggal aktifkan filter list > annoyances > enable Fanboy Annoyances dan Ublock Filter - Annoyances.
