Install Ioncube Di Linux

May 13, 2020 • One minute to read

Pertama seperti biasa cari installan dulu di Aur, tapi sepertinya sudah out-of-date dan baru teringat kalau php yang diinstall menggunakan Xampp. Coba cari tutorial digoogle, kebanyakan menggunakan windows. Tutorial yang saya pakai juga ditujukan untuk xampp di Windows, tapi pada prinsipnya sama dengan linux.

Workflownya:

  • Download ioncube loader dan cocokkan versinya dengan versi php
  • Extract hasil download dan pindahkan file ioncube ke folder /opt/lampp/php/ioncube, buat foldernya jika belum ada
  • Edit php.ini dan tambahkan kode zend_extension=/opt/lampp/php/ioncube/ioncube_loader_lin_7.3.so
  • cek php info, pastikan ada logo zend engine

Untuk tutorial lebih lanjut sudah saya tuliskan disini : https://bash.my.id/cara-install-ioncube-loader-di-xampp-versi-linux.html
