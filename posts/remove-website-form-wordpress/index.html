Cara Remove Website Form di Wordpress

Jan 01, 2021 • One minute to read

Bagi yang males nampilin url field atau form untuk mengisi url di wordpress karena alasan seo atau alasan males kena spam. Bisa menggunakan plugin remove fields atau remove comment website/url box.

Atau

Kalau mau manual bisa menggunakan snippet dibawah ini :

add_filter('comment_form_default_fields', 'website_remove');
function website_remove($fields)
{
if(isset($fields['url']))
unset($fields['url']);
return $fields;
}

Copy snippet diatas lalu taruh di file function.php pada theme folder yang sekarang dipakai.
