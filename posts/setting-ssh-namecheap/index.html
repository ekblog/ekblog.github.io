Setting Ssh di Namecheap

Nov 26, 2020 • One minute to read

Terbeli juga dengan gratis hosting dari namecheap selama sebulan, mulailah acara pindahan. Seperti biasa saya mencari dulu file backupnya. Setelah ketemu mulailah pindahan menggunakan plugin yang backup.

Percobaan pertama backupnya gagal, yang kedua sepertinya juga gagal.

Lanjut cari-cari menu di cpanel namecheap, ada fitur ssh.

Setelah dicoba dan berhasil mengikuti stepnya, nah muncul error seperti ini

Okelah kita coba hubungi supportnya dulu, walaupun ini gratisan yang penting berakhlak…

Btw tutorialnya lengkap setting ssh di namecheap ada disini ya : https://www.namecheap.com/support/knowledgebase/article.aspx/1016/89/how-to-access-a-hosting-account-via-ssh#macos

Setelah kontak livenya ternyata ga perlu hubungi support untuk mengaktifnya cukup mengikuti instruksi disini : https://www.namecheap.com/support/knowledgebase/article.aspx/10040/2210/how-to-enable-ssh-shell-in-cpanel

Buka cpanel trus cari menu “Manage Shell”

Terus aktifkan enable ssh access

Intinya

  1. Siapin pub key, bisa generate di cpanel atau generate di pc.
  2. Enable ssh di menu manage shell seperti gambar diatas
  3. Upload pub key ke menu ssh
  4. Authorize pub key tadi di menu manage
  5. Portnya 21098 untuk shared hosting dan 22 untuk vps atau dedi server (kita pake yang 21098 karena shared hosting)
  6. koneksi ke ssh
ssh userhosting@alamatserver -p21098

Semoga berhasil dan dapat gratisan sebulannya.
