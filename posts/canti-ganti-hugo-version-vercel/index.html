Canti Ganti Hugo Version Vercel

Aug 06, 2021 • One minute to read

Tadi waktu mau deploy blog baru putraeka.eu.org di vercel, ada kesalahan di buildnya sehingga tidak bisa terdeploy.

 unable to parse translation #0 because invalid plural category title

Yang jadi masalah adalah sebenarnya kode error diatasnya your hugo version is not compatible

Search google akhirnya nemu cara mengganti versi hugo di vercel

Buat file di root site dengan nama vercel.json

Isi dengan perintah berikut

{
 "build": {
   "env": {
     "HUGO_VERSION": "0.85.0"
   }
 }
}

Ganti versi sesuai dengan yang berhasil digunakan di localhost.
